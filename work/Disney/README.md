#### 迪士尼介绍: 

*迪士尼乐园于1955年7月开园，立刻成为世界上最具知名度和人气的主题公园。由[华特·迪士尼]创办，至2016年底共在全世界开设6个度假区。*很多人可能不知道，“迪士尼乐园“其实是”迪士尼度假区“中的一个部分。除了乐园外，通常情况下，“迪士尼度假区“一般还包括主题酒店、迪士尼小镇和一系列休闲娱乐设施。而我们常说的”迪士尼乐园“里则包含许多”主题园区“，不同主题园区内则有不同游乐设施。
